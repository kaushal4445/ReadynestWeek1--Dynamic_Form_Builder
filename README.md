## 📌 Project Overview

Dynamic Form Builder is a powerful form management platform that enables users to create custom forms without coding. Users can collect responses, analyze submissions, share forms through QR codes, and manage everything from a beautiful dashboard.

This project was developed as part of my **Full Stack Development Internship at ReadyNest Corporation**.

---

## ✨ Features

- 🔐 User Authentication (Signup/Login)
- 📝 Create & Manage Dynamic Forms
- 🎯 Multiple Input Types
  - Text
  - Email
  - Number
  - Date
  - Radio Buttons
  - Checkboxes
  - Dropdowns
- 👀 Live Form Preview
- 📊 Response Collection
- 📈 Analytics Dashboard
- 📱 Responsive Design
- 🔗 QR Code Sharing
- ⭐ Premium Plan Integration
- 🛡 JWT Authentication
- ☁ MongoDB Atlas Cloud Storage

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside backend folder:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

RAZORPAY_KEY_ID=your_key_id

RAZORPAY_KEY_SECRET=your_secret
```
